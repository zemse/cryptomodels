## Infrastructure Overview

### ClearNode Role

ClearNodes are network participants that:
- Manage cleared transactions
- Settle disputes
- Maintain network state
- Earn protocol rewards

### Participation Requirements

To run a ClearNode, you will need:
- Sufficient hardware resources
- Network connectivity
- Stake or collateral (amount TBD)
- Compliance with network rules
